# Платформа курсов обучения

Современная платформа для размещения обучающих курсов, созданная на HTML, CSS и JavaScript.

## Структура проекта

```
.
├── index.html          # Главная страница
├── course-n8n.html     # Страница курса по n8n
├── styles.css          # Общие стили
├── script.js           # JavaScript функциональность
├── data/
│   └── n8n/           # Уроки курса по n8n
│       ├── lesson_1.html
│       ├── lesson_2.html
│       └── ... (26 уроков)
└── README.md          # Документация
```

## Особенности

- **Современный минималистичный дизайн** — чистый и понятный интерфейс
- **Адаптивная верстка** — работает на всех устройствах
- **Отслеживание прогресса** — автоматическое сохранение пройденных уроков
- **Подсветка синтаксиса** — для примеров кода используется Prism.js
- **Навигация по урокам** — удобная боковая панель с разделами
- **Прогресс-бар** — визуальное отображение прогресса прохождения курса

## Использование

1. Откройте `index.html` в браузере
2. Выберите курс из списка
3. Начните обучение, выбирая уроки из навигации

## Курс по n8n

Полный курс по n8n включает:

1. **Введение в n8n** (3 урока)
   - Что такое n8n
   - Преимущества и области применения
   - Обзор интерфейса

2. **Установка и настройка** (3 урока)
   - Установка n8n
   - Первоначальная настройка
   - Настройка окружения

3. **Основы работы с n8n** (4 урока)
   - Создание первого workflow
   - Основные ноды
   - Переменные и выражения
   - Обработка ошибок

4. **Работа с LLM и DeepSeek** (4 урока)
   - Настройка DeepSeek API
   - Создание нод для работы с LLM
   - Промпты и обработка ответов
   - Управление контекстом

5. **Мультиагентные системы** (4 урока)
   - Архитектура мультиагентных систем
   - Создание агентов в n8n
   - Коммуникация между агентами
   - Оркестрация workflow

6. **Быстрое прототипирование** (4 урока)
   - Шаблоны для создания агентов
   - Отладка мультиагентных систем
   - Тестирование и валидация
   - Оптимизация производительности

7. **Продвинутые техники** (4 урока)
   - Параллельная обработка
   - Управление состоянием
   - Обработка больших объемов данных
   - Мониторинг и логирование

## Технологии

- **HTML5** — семантическая разметка
- **CSS3** — современные стили с CSS переменными
- **JavaScript (ES6+)** — интерактивность и функциональность
- **Prism.js** — подсветка синтаксиса кода (через CDN)
- **LocalStorage** — сохранение прогресса пользователя

## Функциональность

### Отслеживание прогресса

Прогресс сохраняется в LocalStorage браузера. Каждый завершенный урок отмечается галочкой в навигации.

### Навигация

- Боковая панель с разделами курса
- Раскрывающиеся секции
- Активный урок выделяется
- Кнопки "Предыдущий/Следующий урок"

### Адаптивность

Сайт полностью адаптирован для:
- Десктопов
- Планшетов
- Мобильных устройств

## Разработка

Для локальной разработки просто откройте файлы в браузере. Для production рекомендуется использовать веб-сервер.

### Запуск локального сервера

```bash
# Python
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

Затем откройте `http://localhost:8000` в браузере.

## Добавление новых курсов

1. Создайте новую страницу курса (например, `course-new.html`)
2. Добавьте структуру курса в `script.js`
3. Создайте папку с уроками в `data/`
4. Добавьте карточку курса на главную страницу
