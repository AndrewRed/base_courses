Установка n8n

Существует несколько способов установки n8n. Выберите наиболее подходящий для вашего случая.

1. Установка через Docker (Рекомендуется)

Docker — самый простой и надежный способ установки n8n.

Базовая установка

docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  n8nio/n8n

Установка с постоянным хранением данных

docker run -d \
  --name n8n \
  -p 5678:5678 \
  -v ~/.n8n:/home/node/.n8n \
  -e N8N_BASIC_AUTH_ACTIVE=true \
  -e N8N_BASIC_AUTH_USER=admin \
  -e N8N_BASIC_AUTH_PASSWORD=your_password \
  n8nio/n8n

Использование Docker Compose

Создайте файл docker-compose.yml:

version: '3.8'

services:
  n8n:
    image: n8nio/n8n
    container_name: n8n
    ports:
      - "5678:5678"
    environment:
      - N8N_BASIC_AUTH_ACTIVE=true
      - N8N_BASIC_AUTH_USER=admin
      - N8N_BASIC_AUTH_PASSWORD=your_secure_password
      - N8N_HOST=localhost
      - N8N_PORT=5678
      - N8N_PROTOCOL=http
    volumes:
      - ~/.n8n:/home/node/.n8n
    restart: unless-stopped

Запуск:

docker-compose up -d

2. Установка через npm

Если у вас установлен Node.js, можно установить n8n глобально:

npm install n8n -g

Запуск:

n8n start

3. Облачная версия

n8n также доступен в облаке на n8n.io. Это удобно для быстрого старта, но имеет ограничения бесплатного плана.

4. Установка на сервер

Требования

  • Node.js 18.x или выше
  • Минимум 2GB RAM
  • Свободное место на диске для данных

Шаги установки

# Установка Node.js (если не установлен)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt-get install -y nodejs

# Установка n8n
sudo npm install n8n -g

# Создание systemd service
sudo nano /etc/systemd/system/n8n.service

Содержимое файла service:

[Unit]
Description=n8n workflow automation
After=network.target

[Service]
Type=simple
User=your_user
ExecStart=/usr/bin/n8n start
Restart=on-failure

[Install]
WantedBy=multi-user.target

Запуск сервиса:

sudo systemctl enable n8n
sudo systemctl start n8n

Проверка установки

После установки откройте браузер и перейдите по адресу:

http://localhost:5678

Вы должны увидеть страницу приветствия n8n, где можно создать учетную запись.

Рекомендации

  • Для разработки и тестирования используйте Docker
  • Для production используйте Docker Compose с правильной настройкой безопасности
  • Обязательно настройте базовую аутентификацию для production
  • Настройте регулярное резервное копирование данных
