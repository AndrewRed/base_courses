Обработка больших объемов данных

Обработка больших объемов данных требует специальных техник. Изучим подходы к работе с большими данными в n8n.

1. Разбиение на батчи

Обработка данных порциями:

Input (большой массив) →
  Split In Batches (размер: 100) →
  Agent (обработка батча) →
  Merge →
  Aggregation

2. Стриминг данных

Обработка данных по мере поступления:

Stream Source →
  Code (обработка чанка) →
  Agent (обработка) →
  Stream Sink

3. Параллельная обработка

Input →
  Split In Batches →
    Split (параллельная обработка батчей) →
      ├─ Agent 1
      ├─ Agent 2
      └─ Agent 3
    → Merge
  → Merge →
  Final Aggregation

4. Фильтрация перед обработкой

Input →
  Code (фильтрация) →
  Agent (обработка только нужных данных)

5. Агрегация на лету

Input →
  Split In Batches →
  Code (частичная агрегация) →
  Merge →
  Code (финальная агрегация)

6. Best Practices

  • Используйте батчинг для больших объемов
  • Параллелизуйте обработку
  • Фильтруйте данные перед обработкой
  • Используйте стриминг где возможно
  • Мониторьте использование памяти
