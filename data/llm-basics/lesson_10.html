Примеры эффективных промптов

В этом уроке мы рассмотрим конкретные примеры эффективных промптов для разных ситуаций. Эти примеры вы сможете использовать как основу для своих задач, адаптировав их под свои потребности.

Примеры для учителей

Пример 1: Создание плана урока

Промпт:

Я учитель математики, готовлю урок для 5 класса на тему "Дроби". У меня есть 45 минут. Класс состоит из 28 учеников с разным уровнем подготовки.

Создай подробный план урока, который включает:

  • Введение в тему (5 минут) — как заинтересовать учеников
  • Объяснение материала (15 минут) — простые примеры и объяснения
  • Практические задания (20 минут) — задания разной сложности
  • Подведение итогов и домашнее задание (5 минут)

План должен быть структурирован по времени, включать конкретные примеры для объяснения и задания для трех уровней: базовый, средний и продвинутый.

Пример 2: Создание вопросов для викторины

Промпт:

Создай 15 вопросов для викторины по географии России для учеников 8 класса. Вопросы должны быть:

  • Разной сложности: 5 легких, 7 средних, 3 сложных
  • С вариантами ответов (4 варианта для каждого вопроса)
  • С правильными ответами в конце
  • Покрывать разные темы: рельеф, климат, реки, города, природные зоны

Представь в виде таблицы с колонками: номер вопроса, вопрос, варианты ответов (A, B, C, D), правильный ответ.

Пример 3: Объяснение сложной темы

Промпт:

Объясни тему "Фотосинтез" для учеников 5 класса. Текст должен быть:

  • Понятным для детей 10-11 лет
  • Включать простые аналогии из повседневной жизни
  • Содержать 3-4 ключевых факта
  • Быть увлекательным, но научно точным

Используй простые слова, избегай сложных терминов. Если термин необходим, объясни его простыми словами.

Примеры для работников культуры

Пример 1: Описание выставки

Промпт:

Напиши описание выставки "Русские художники 19 века" для посетителей музея. Текст должен:

  • Быть интересным и доступным для широкой аудитории
  • Включать информацию о 5-7 ключевых экспонатах
  • Рассказывать об историческом контексте эпохи
  • Быть длиной примерно 300-400 слов

Стиль: увлекательный, но уважительный. Избегай сложных искусствоведческих терминов без объяснений.

Пример 2: Программа культурного мероприятия

Промпт:

Создай программу литературного вечера, посвященного творчеству Анны Ахматовой. Мероприятие рассчитано на взрослую аудиторию, продолжительность — 2 часа.

Программа должна включать:

  • Вступительное слово ведущего (10 минут)
  • Чтение стихов профессиональными чтецами (40 минут)
  • Лекцию о жизни и творчестве поэтессы (30 минут)
  • Музыкальную паузу (15 минут)
  • Обсуждение и вопросы аудитории (20 минут)
  • Заключительное слово (5 минут)

Представь в виде таблицы с указанием времени для каждого блока и кратким описанием содержания.

Пример 3: Текст для афиши

Промпт:

Напиши текст для афиши театрального спектакля "Ревизор" по пьесе Гоголя. Текст должен:

  • Быть кратким (100-150 слов)
  • Заинтересовывать потенциальных зрителей
  • Включать основную информацию: дата, время, место
  • Рассказывать о содержании спектакля, но не раскрывать сюжет полностью

Стиль: яркий, привлекательный, но не перегруженный деталями.

Универсальные примеры

Пример 1: Перевод и адаптация текста

Промпт:

Переведи следующий текст с английского на русский, адаптировав его для русскоязычной аудитории:

[Вставить текст]

Требования к переводу:

  • Сохранить смысл и стиль оригинала
  • Адаптировать культурные ссылки для русской аудитории
  • Использовать естественный русский язык
  • Сохранить структуру и форматирование

Пример 2: Создание списка

Промпт:

Создай список из 10 способов заинтересовать учеников на уроке истории. Каждый способ должен:

  • Быть конкретным и практичным
  • Включать краткое описание (1-2 предложения)
  • Быть применимым для разных возрастов

Представь в виде нумерованного списка с краткими описаниями.

Пример 3: Редактирование текста

Промпт:

Отредактируй следующий текст, улучшив его:

[Вставить текст]

Внеси следующие изменения:

  • Исправь грамматические и орфографические ошибки
  • Улучши стиль и читаемость
  • Сделай текст более структурированным
  • Сохрани основной смысл и тон

В конце укажи, какие изменения были внесены.

Как использовать эти примеры

  1. Скопируйте подходящий пример — выберите промпт, наиболее близкий к вашей задаче
  2. Адаптируйте под свою ситуацию — измените детали (класс, тему, аудиторию и т.д.)
  3. Добавьте свои требования — включите специфические детали вашей задачи
  4. Протестируйте — используйте промпт и посмотрите на результат
  5. Улучшите при необходимости — если результат не идеален, уточните промпт

Практическое задание

Выберите одну из ваших реальных задач и создайте промпт, используя один из примеров как основу. Затем:

  1. Используйте промпт в chat.qwen.ai
  2. Оцените результат
  3. Если нужно, улучшите промпт и попробуйте снова
  4. Сравните результаты первого и улучшенного промптов
