Практические задания по созданию промптов

В этом уроке мы закрепим все изученное о промптах на практике. Вы выполните несколько заданий, которые помогут вам уверенно создавать эффективные промпты для реальных задач.

Задание 1: Создание плана урока

Ситуация

Вы учитель биологии. Вам нужно подготовить урок для 7 класса на тему "Клетка — основная единица жизни". Урок длится 45 минут. В классе 30 учеников с разным уровнем подготовки.

Ваша задача

Создайте промпт для QWEN, который поможет вам получить подробный план урока. Промпт должен включать:

  • Роль (кто вы)
  • Контекст (класс, тема, время, количество учеников)
  • Задачу (что нужно создать)
  • Требования (что должно быть в плане)
  • Формат (как представить результат)

Пример решения

Я учитель биологии, готовлю урок для 7 класса на тему "Клетка — основная единица жизни". У меня есть 45 минут, в классе 30 учеников с разным уровнем подготовки.

Создай подробный план урока, который включает:

  • Введение в тему (5 минут) — как заинтересовать учеников
  • Объяснение материала о строении клетки (20 минут) — простые и понятные объяснения
  • Практическое задание (15 минут) — работа с микроскопом или схемами
  • Закрепление и домашнее задание (5 минут)

План должен быть структурирован по времени, включать конкретные примеры для объяснения и задания для разных уровней подготовки. Представь в виде таблицы с колонками: время, этап урока, описание, материалы.

Попробуйте сами

Используйте этот пример как основу, но адаптируйте под свой предмет и тему. Затем используйте промпт в chat.qwen.ai и оцените результат.

Задание 2: Создание программы мероприятия

Ситуация

Вы работник культуры, готовите литературный вечер, посвященный творчеству Сергея Есенина. Мероприятие для взрослой аудитории, продолжительность 2 часа.

Ваша задача

Создайте промпт для создания программы мероприятия. Программа должна включать разные форматы: чтение стихов, лекцию, музыкальные паузы, обсуждение.

Подсказки

  • Укажите целевую аудиторию
  • Опишите желаемые форматы
  • Укажите общую продолжительность
  • Попросите распределить время между блоками

Попробуйте создать промпт

Напишите свой промпт, используя структуру из предыдущих уроков. Затем протестируйте его в chat.qwen.ai.

Задание 3: Объяснение сложной темы

Ситуация

Вам нужно объяснить ученикам 6 класса сложную тему из вашего предмета простым языком.

Ваша задача

Выберите сложную тему из вашего предмета и создайте промпт, который поможет получить простое объяснение для детей.

Требования к промпту

  • Укажите возраст аудитории
  • Попросите использовать простые слова
  • Попросите включить примеры из повседневной жизни
  • Укажите желаемую длину объяснения

Пример темы

Если вы не знаете, какую тему выбрать, попробуйте:

  • Для истории: "Причины и последствия крещения Руси"
  • Для биологии: "Как работает пищеварительная система"
  • Для физики: "Что такое электрический ток"
  • Для литературы: "Что такое метафора и зачем она нужна"

Задание 4: Работа с режимом SEARCH

Ситуация

Вам нужна актуальная информация по теме, связанной с вашей работой.

Ваша задача

Создайте два промпта для одного и того же вопроса:

  1. Промпт в обычном режиме
  2. Промпт с использованием режима SEARCH

Пример вопроса

Выберите вопрос, требующий актуальной информации, например:

  • "Какие последние изменения в системе образования произошли в этом году?"
  • "Какие новые выставки открылись в крупных музеях России?"
  • "Какие новые методики преподавания появились в последнее время?"

Сравнение результатов

После получения ответов сравните:

  • Какой ответ более актуальный?
  • В каком ответе есть ссылки на источники?
  • Какой ответ кажется более точным?

Задание 5: Улучшение плохого промпта

Плохой промпт

"Сделай что-то про урок"

Ваша задача

Улучшите этот промпт, добавив:

  • Контекст (кто вы, для кого, какая тема)
  • Конкретную задачу
  • Требования к результату
  • Формат ответа

Сравните результаты

Сначала используйте плохой промпт, затем улучшенный. Сравните результаты и подумайте, что изменилось.

Задание 6: Создание промпта для вашей реальной задачи

Ваша задача

Подумайте о реальной задаче из вашей работы, которую вы хотели бы решить с помощью QWEN. Это может быть:

  • Подготовка материала для урока
  • Создание программы мероприятия
  • Написание текста
  • Создание викторины или заданий
  • Любая другая задача, связанная с текстами

Создайте промпт

Используя все знания из предыдущих уроков, создайте эффективный промпт для вашей задачи. Убедитесь, что промпт включает все элементы хорошей структуры.

Протестируйте и улучшите

  1. Используйте промпт в chat.qwen.ai
  2. Оцените результат
  3. Если нужно, улучшите промпт
  4. Попробуйте снова
  5. Сравните результаты

Чек-лист для самопроверки

Проверьте свой промпт по этому списку:

  • ✓ Есть ли роль или контекст о том, кто вы?
  • ✓ Указана ли целевая аудитория (для кого создается материал)?
  • ✓ Четко ли сформулирована задача?
  • ✓ Есть ли конкретные требования к результату?
  • ✓ Указан ли желаемый формат ответа?
  • ✓ Нет ли противоречий в инструкциях?
  • ✓ Не слишком ли много задач в одном промпте?
  • ✓ Для актуальной информации используется ли режим SEARCH?

Советы для практики

  • Начинайте с простых задач — постепенно переходите к сложным
  • Экспериментируйте — пробуйте разные формулировки
  • Сохраняйте удачные промпты — они могут пригодиться позже
  • Учитесь на ошибках — если промпт не сработал, подумайте, что можно улучшить
  • Практикуйтесь регулярно — чем больше практики, тем лучше результаты
