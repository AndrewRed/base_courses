Создание шаблонов и оптимизация процессов

В этом финальном уроке раздела мы научимся создавать шаблоны промптов и оптимизировать рабочие процессы, чтобы максимально эффективно использовать QWEN в ежедневной работе учителя.

Зачем создавать шаблоны?

Шаблоны промптов помогают:

  • Экономить время — не нужно каждый раз придумывать промпт заново
  • Получать стабильные результаты — проверенные формулировки дают предсказуемые результаты
  • Стандартизировать процессы — одинаковый подход к похожим задачам
  • Улучшать качество — оттачивая шаблоны, вы получаете лучшие результаты

Создание библиотеки шаблонов

Шаблон 1: Планирование урока

Название шаблона: "План урока"

Шаблон промпта:

"Я учитель [ПРЕДМЕТ] для [КЛАСС] класса. Создай подробный план урока на [ВРЕМЯ] минут по теме '[ТЕМА]'. План должен включать:

  • Введение в тему ([ВРЕМЯ] минут)
  • Объяснение материала ([ВРЕМЯ] минут)
  • Практическое задание ([ВРЕМЯ] минут)
  • Подведение итогов и домашнее задание ([ВРЕМЯ] минут)

Включи конкретные примеры, интерактивные элементы и задания для разных уровней подготовки. Представь в виде таблицы с временными метками."

Как использовать: Замените [ПРЕДМЕТ], [КЛАСС], [ВРЕМЯ], [ТЕМА] на конкретные значения.

Шаблон 2: Создание заданий

Название шаблона: "Задания для урока"

Шаблон промпта:

"Создай [КОЛИЧЕСТВО] заданий по теме '[ТЕМА]' для [КЛАСС] класса. Задания должны быть разной сложности: [X] легких, [Y] средних, [Z] сложных. Включи ответы и краткие объяснения."

Как использовать: Замените [КОЛИЧЕСТВО], [ТЕМА], [КЛАСС], [X], [Y], [Z] на конкретные значения.

Шаблон 3: Проверка работы

Название шаблона: "Анализ работы ученика"

Шаблон промпта:

"Проанализируй эту работу ученика [КЛАСС] класса на тему '[ТЕМА]'. Оцени:

  • Соответствие теме
  • Структуру и логику
  • Грамматику и стиль
  • Содержание

Укажи сильные стороны и 2-3 конкретные области для улучшения. Создай конструктивную обратную связь для ученика."

Как использовать: Загрузите работу, замените [КЛАСС] и [ТЕМА].

Шаблон 4: Письмо родителям

Название шаблона: "Информационное письмо родителям"

Шаблон промпта:

"Напиши письмо родителям учеников [КЛАСС] класса о [СОБЫТИЕ/СИТУАЦИЯ]. Письмо должно быть:

  • Кратким и информативным
  • Вежливым и профессиональным
  • С четкой структурой
  • С указанием конкретных дат: [ДАТЫ]

Используй официально-деловой стиль, но дружелюбный тон."

Как использовать: Замените [КЛАСС], [СОБЫТИЕ/СИТУАЦИЯ], [ДАТЫ].

Как организовать шаблоны

Способ 1: Сохранение в диалогах

Создайте отдельный диалог "Шаблоны промптов" и сохраните там все ваши шаблоны. Когда нужно, просто скопируйте шаблон и адаптируйте.

Способ 2: Документ с шаблонами

Создайте документ (Word, Google Docs) с вашими шаблонами. Организуйте их по категориям:

  • Планирование уроков
  • Создание материалов
  • Проверка работ
  • Работа с родителями
  • Документация

Способ 3: Комбинированный подход

Используйте оба способа: сохраняйте часто используемые шаблоны в диалогах, а полную библиотеку — в документе.

Оптимизация рабочих процессов

Процесс 1: Подготовка к неделе

День 1 (воскресенье):

  1. Создайте план недели с помощью QWEN
  2. Распределите темы по дням
  3. Определите, какие материалы нужны

День 2-6 (понедельник-пятница):

  1. Утром: создайте план урока на день
  2. Подготовьте материалы (задания, примеры)
  3. После уроков: проверьте работы с помощью QWEN

Процесс 2: Проверка работ

Оптимизированный процесс:

  1. Соберите все работы
  2. Загрузите работы в QWEN (по одной или группами)
  3. Используйте шаблон для анализа
  4. Получите обратную связь
  5. Просмотрите и адаптируйте под каждого ученика
  6. Запишите комментарии

Экономия времени: Вместо 2-3 минут на каждую работу, вы тратите 30 секунд на адаптацию готовой обратной связи.

Процесс 3: Подготовка к родительскому собранию

За неделю до собрания:

  1. Создайте структуру выступления
  2. Подготовьте материалы по каждой теме
  3. Создайте ответы на возможные вопросы

За день до собрания:

  1. Просмотрите и адаптируйте материалы
  2. Добавьте конкретные примеры из вашего класса
  3. Подготовьте визуальные материалы (если нужно)

Создание комплексных рабочих процессов

Пример: Полная подготовка урока

Шаг 1: Планирование

"Создай план урока по теме '[ТЕМА]' для [КЛАСС] класса"

Шаг 2: Материалы

"Создай задания для практической части этого урока"

"Создай вопросы для проверки понимания"

Шаг 3: Домашнее задание

"Создай домашнее задание на основе этого урока"

Шаг 4: Визуальные материалы

"Создай иллюстрацию для этого урока: [описание]"

Результат: Полный комплект материалов для урока за 10-15 минут вместо часа.

Советы по оптимизации

  • Используйте один диалог для одной задачи — так контекст сохраняется
  • Создавайте шаблоны постепенно — не пытайтесь создать все сразу
  • Улучшайте шаблоны — если результат не идеален, доработайте шаблон
  • Организуйте шаблоны — так их легче найти и использовать
  • Делитесь с коллегами — обменивайтесь удачными шаблонами
  • Адаптируйте под себя — каждый учитель работает по-своему

Типичные ошибки при создании шаблонов

  • Слишком общие шаблоны — "Создай план урока" не даст хорошего результата
  • Недостаточно переменных — не указываете, что нужно заменить
  • Слишком сложные шаблоны — лучше разбить на несколько простых
  • Не обновляете шаблоны — со временем они могут устареть

Практическое задание

Создайте свою библиотеку шаблонов:

  1. Выберите 3-5 задач, которые вы выполняете регулярно
  2. Создайте шаблоны промптов для каждой задачи
  3. Протестируйте шаблоны на реальных задачах
  4. Улучшите шаблоны на основе результатов
  5. Организуйте шаблоны (сохраните в диалоге или документе)
  6. Используйте шаблоны в работе и оцените экономию времени
