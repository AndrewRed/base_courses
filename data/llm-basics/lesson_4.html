Интерфейс и основные элементы

В этом уроке мы подробно разберем интерфейс chat.qwen.ai и научимся ориентироваться во всех его элементах. Это поможет вам уверенно работать с системой.

Главный экран

После входа в chat.qwen.ai вы увидите главный экран. Давайте разберем его основные части:

1. Боковое меню (слева)

Левая панель содержит:

  • Логотип Qwen — вверху слева, фиолетово-белая эмблема с текстом "Qwen"
  • Кнопка "+ Новый чат" — фиолетовая кнопка для начала нового разговора
  • Поиск в чатах — поле поиска с иконкой лупы "Поиск в чатах" для поиска по истории диалогов
  • Все чаты — раздел с историей всех ваших диалогов, организованных по времени:
    • "Сегодня" — диалоги за сегодняшний день
    • "Вчера" — диалоги за вчерашний день
    • "Предыдущие 7 дней" — диалоги за последнюю неделю
  • Профиль пользователя — внизу панели отображается ваш профиль с именем

2. Область диалога (центр)

Основная рабочая область включает:

  • Выбор модели — вверху выпадающий список с выбранной моделью (например, "Qwen3-Max")
  • Приветствие — персональное приветствие, например "Добрый вечер, [Ваше имя]"
  • Поле ввода — большое округлое поле в центре с плейсхолдером "Чем я могу помочь вам сегодня?"
    • В правом нижнем углу поля ввода есть иконка микрофона для голосового ввода
  • Кнопки режимов работы — над полем ввода расположены кнопки:
    • "+" — для добавления контента (файлов, изображений)
    • "Мышление" — режим мышления с выпадающим меню
    • "Поиск" — кнопка с иконкой глобуса для включения режима поиска в интернете
  • Чипы-подсказки — под полем ввода расположены кнопки-подсказки для быстрого доступа к функциям:
    • "Редактирование изображения" — для работы с изображениями
    • "веб-разработка" — для задач веб-разработки
    • "Учиться" — для обучения
    • "Глубокое исследование" — для режима глубокого исследования
    • "Генерация изображений" — для создания изображений
    • "Генерация видео" — для создания видео
    • "Больше" — для дополнительных опций
  • Кнопка "Снимок экрана" — внизу слева для создания скриншотов

3. Как использовать основные функции

Для быстрого доступа к функциям используйте чипы-подсказки под полем ввода или кнопки над ним:

  • Генерация изображений — нажмите на чип "Генерация изображений" или используйте кнопку "+" для добавления запроса на генерацию
  • Глубокое исследование — нажмите на чип "Глубокое исследование" или кнопку "Поиск" для включения режима поиска в интернете
  • Загрузка файлов — используйте кнопку "+" для прикрепления документов
  • Голосовой ввод — нажмите на иконку микрофона в правом нижнем углу поля ввода
  • Режим мышления — используйте кнопку "Мышление" для выбора режима работы модели

Основные действия

Как задать вопрос

  1. Кликните в поле ввода (обычно внизу экрана)
  2. Напишите свой вопрос или запрос
  3. Нажмите кнопку отправки или клавишу Enter
  4. Дождитесь ответа QWEN

Как начать новый диалог

Есть несколько способов:

  • Нажать кнопку "+ Новый чат" в левой боковой панели (фиолетовая кнопка под логотипом Qwen)
  • Выбрать существующий диалог из истории в левой панели
  • Использовать горячую клавишу (если доступна)

Как сохранить диалог

Обычно диалоги сохраняются автоматически. Но вы также можете:

  • Дать диалогу название (если есть такая опция)
  • Добавить его в избранное
  • Экспортировать диалог (если доступно)

Индикаторы и подсказки

Обращайте внимание на:

  • Индикатор загрузки — показывает, что QWEN обрабатывает ваш запрос
  • Статус подключения — показывает, подключены ли вы к серверу
    • Подсказки — часто интерфейс предлагает примеры вопросов или действий

Адаптивность интерфейса

Интерфейс chat.qwen.ai может выглядеть немного по-разному в зависимости от:

  • Размера экрана (компьютер, планшет, телефон)
  • Версии сайта (интерфейс может обновляться)
  • Языка интерфейса

Не переживайте, если ваш интерфейс выглядит немного иначе — основные элементы остаются теми же.

Практическое задание

Попробуйте выполнить следующие действия:

  1. Откройте chat.qwen.ai и войдите в систему
  2. Найдите все основные элементы интерфейса, о которых мы говорили
  3. Начните новый диалог
  4. Задайте простой вопрос и посмотрите, как работает интерфейс
  5. Попробуйте найти настройки (обычно в боковом меню или в правом верхнем углу)

Советы

  • Изучите интерфейс постепенно — не нужно запоминать всё сразу
  • Экспериментируйте — лучший способ изучить интерфейс — это использовать его
  • Обращайте внимание на подсказки — интерфейс часто дает полезные советы
  • Не бойтесь ошибиться — вы всегда можете начать заново
