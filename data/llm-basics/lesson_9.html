Структура хорошего промпта

Хороший промпт — это не просто вопрос, это структурированная инструкция. В этом уроке мы разберем, из каких элементов состоит эффективный промпт и как их правильно сочетать.

Основные элементы хорошего промпта

1. Роль (Role)

Укажите, кем должен "представляться" QWEN или для кого предназначен ответ.

Пример: "Ты опытный учитель истории. Помоги мне..."

Или:

Пример: "Напиши текст для детей 7-8 лет о..."

2. Контекст (Context)

Опишите ситуацию, в которой вы находитесь.

Пример: "Я готовлю урок для 6 класса по теме 'Древний Рим'. У меня есть 45 минут. Класс состоит из 25 учеников с разным уровнем подготовки."

3. Задача (Task)

Четко сформулируйте, что нужно сделать.

Пример: "Создай план урока, который включает объяснение материала, интерактивное задание и домашнюю работу."

4. Требования (Requirements)

Укажите конкретные требования к результату.

Пример: "План должен быть структурирован по времени, включать ключевые даты и события, и содержать задания для разных уровней подготовки."

5. Формат (Format)

Укажите, в каком формате вы хотите получить ответ.

Пример: "Представь результат в виде структурированного плана с временными метками для каждого этапа урока."

Полный пример хорошего промпта

Роль: Ты опытный учитель литературы, специализирующийся на работе с детьми.

Контекст: Я готовлю литературный вечер для учеников 8-9 классов, посвященный творчеству Пушкина. Мероприятие будет проходить в актовом зале школы, продолжительность — 1,5 часа. Ожидается около 50 учеников.

Задача: Создай программу литературного вечера, которая будет интересна подросткам и познакомит их с творчеством Пушкина.

Требования: Программа должна включать: - Вступительное слово (5-7 минут) - Чтение стихов учениками (20 минут) - Интерактивный блок с викториной (15 минут) - Музыкальные паузы (2 паузы по 5 минут) - Заключительное слово (5 минут)

Формат: Представь программу в виде таблицы с указанием времени для каждого блока и кратким описанием содержания.

Шаблоны для разных задач

Роль: [Кто вы или кем должен быть QWEN]
Контекст: [Для кого, какой класс, какая тема]
Задача: [Что нужно создать]
Требования: [Что должно быть включено]
Формат: [В каком виде представить результат]

Техники улучшения промптов

1. Используйте примеры

Иногда полезно показать QWEN пример того, что вы хотите:

"Создай список вопросов для викторины по географии. Пример хорошего вопроса: 'Какая самая длинная река в России?'"

2. Используйте отрицания

Укажите, чего НЕ должно быть в ответе:

"Напиши текст для школьной газеты. Не используй сложные термины и длинные предложения."

3. Задавайте структуру

Попросите организовать ответ определенным образом:

"Представь информацию в следующем порядке: 1) Краткое введение, 2) Основные факты, 3) Интересные детали, 4) Заключение."

4. Используйте итерации

Начните с базового промпта, затем улучшайте его на основе ответа:

Первый промпт: "Напиши план урока"

Второй промпт (после получения ответа): "Сделай план более детальным, добавь временные метки для каждого этапа"

Практическое задание

Создайте хороший промпт для одной из следующих задач, используя все элементы структуры:

  1. Создание программы культурного мероприятия для музея
  2. Подготовка плана урока с интерактивными элементами
  3. Написание статьи для школьной газеты
  4. Создание викторины по определенной теме

Для каждой задачи укажите: роль, контекст, задачу, требования и формат.

Частые ошибки в структуре

  • Отсутствие контекста — QWEN не знает, для кого и зачем создается материал
  • Неопределенная задача — "сделай что-то хорошее" не даст хорошего результата
  • Отсутствие формата — ответ может быть не в том виде, который вам нужен
  • Слишком много требований сразу — лучше разбить на несколько промптов
  • Противоречивые инструкции — "кратко, но подробно" запутает модель
